## 技術スタック
+ フロントエンド: Nuxt3
+ バックエンド: Rails
+ データベース: PostgreSQL
+ インフラストラクチャ: AWS

### インフラストラクチャ(AWS)
+ CloudWatch: ログ監視
+ CodePipeline: コードデプロイ
+ ECR: コンテナイメージレジストリ
+ ECS: コンテナオーケストレーション
+ ELB: ロードバランサー
+ IAM: アクセス管理
+ RDS: リレーショナルデータベース
+ S3: オブジェクトストレージ
